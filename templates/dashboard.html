Guido van Rossum

Founder of Python

Guido van Rossum is a Dutch programmer best known as the creator of the Python programming language, for which he was the "benevolent dictator for life" until he stepped down from the position in July 2018.

James Gosling

Founder of Java

James Arthur Gosling, often referred to as "Dr. Java," OC is a Canadian computer scientist, best known as the founder and lead designer behind the Java programming language.

Dennis Ritchie

Founder of C Programming

Dennis MacAlistair Ritchie was an American computer scientist. He created the C programming language and, with long-time colleague Ken Thompson, the Unix operating system and B programming language.
